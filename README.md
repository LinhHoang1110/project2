## **Project 2**
Yêu cầu: 
- Cắt psd thành các thành phần
-  Xác định font chữ sử dụng : nguồn lấy font, kiểu, các mức độ bold cần có, các font size sẽ sử dụng, .... Xác định các màu sẽ sử dụng
